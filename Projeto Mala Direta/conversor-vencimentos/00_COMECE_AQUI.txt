🎯 MINHA PRIMEIRA CONVERSÃO
═════════════════════════════════════════════════════════════════════════

Suponha que seu PDF se chama "vencimentos_maio.pdf":

1. Coloque "vencimentos_maio.pdf" nesta pasta

2. Abra terminal e execute:

   Windows:
   > converter_vencimentos.bat vencimentos_maio.pdf

   macOS/Linux:
   $ ./converter_vencimentos.sh vencimentos_maio.pdf

3. Quando terminar, você terá:
   ✅ vencimentos_maio_vencimentos.xlsx

4. Abra em Excel/Calc/Sheets:
   ✓ Dados formatados
   ✓ Cores por tipo
   ✓ Filtros ativos
   ✓ Pronto para usar!

🐛 ALGO DEU ERRADO?
═════════════════════════════════════════════════════════════════════════

"Python não encontrado"
→ Instale Python 3.7+ de https://www.python.org/downloads/

"módulo não encontrado (openpyxl/pypdf)"
→ Execute: pip install -r requirements.txt

"arquivo PDF não encontrado"
→ Coloque o PDF na mesma pasta ou use caminho completo

"Permissão negada" (macOS/Linux)
→ Execute: chmod +x converter_vencimentos.sh
