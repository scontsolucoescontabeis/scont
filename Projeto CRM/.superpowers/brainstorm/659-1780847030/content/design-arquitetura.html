Pontos-chave da arquitetura

  • Dev bypass token — variável de ambiente DEV_BYPASS_TOKEN no Supabase. O webhook pula o HMAC da Meta quando recebe o header correto. Removível em produção.
  • Chatbot funciona de verdade — como a mensagem passa pelo pipeline completo do webhook, o chatbot processor roda normalmente e insere as respostas no banco.
  • Meta API falha silenciosamente — o webhook tenta enviar a resposta para o número fake, falha, mas a mensagem BOT/AGENTE já está no banco. O simulador lê via Realtime.
  • Realtime com anon key — nova migração 011_dev_simulator.sql adiciona policy SELECT permissiva nas tabelas mensagens e conversas para o role anon.
  • Isolado de produção — nenhuma mudança de schema em conversas ou mensagens. Contatos com prefixo SIM- nos nomes são facilmente distinguíveis.
