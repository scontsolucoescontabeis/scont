Funcionalidades extras do chatbot

Selecione todas que deseja incluir (multi-seleção)

🔴 Desligar chatbot (atendimento humano) Admin
Chave no painel admin para desativar o bot globalmente ou por departamento. Quando desligado, cliente vai direto para fila humana.
🌙 Horário de atendimento Essencial
Define horário de funcionamento por departamento. Fora do horário, bot avisa e oferece opção de deixar mensagem para retorno.
👤 "Falar com atendente" a qualquer momento Essencial
Em qualquer etapa do fluxo, cliente pode digitar "0" ou clicar em opção para pular o bot e ir direto para atendimento humano.
⏱️ Timeout de inatividade Essencial
Se o cliente não responder em X minutos durante o fluxo do bot, a conversa é encerrada automaticamente com mensagem de aviso.
⭐ Avaliação do atendimento (CSAT) Plus
Ao encerrar a conversa, o bot envia automaticamente uma pesquisa de satisfação de 1 a 5 estrelas. Resultados aparecem nas métricas.
🔁 Reconhecimento de cliente recorrente Plus
Se o cliente já teve atendimento anterior no mesmo departamento, o bot pergunta se é sobre o mesmo assunto ou novo — reduz etapas.
📋 Confirmação antes de rotear Essencial
Antes de transferir, o bot resume a escolha: "Entendido — Dep. Pessoal › Folha › Recálculo. Confirmado?" Evita roteamentos errados.
🚫 Limite de tentativas inválidas Essencial
Se o cliente enviar 3 respostas inválidas (fora do menu), o bot transfere automaticamente para um agente humano.
📝 Coleta de protocolo anterior Plus
Se o cliente menciona um atendimento anterior, o bot coleta o número do protocolo para o agente já ter o contexto ao assumir.
🏷️ Tag automática via chatbot Plus
A conversa é automaticamente tagueada com o departamento e motivo selecionados no bot (ex: "Folha", "Rescisão"), facilitando filtros.
📣 Mensagem de fila de espera Essencial
Após o roteamento, informa ao cliente sua posição na fila e tempo médio de espera estimado para o departamento.
⚙️ Editor visual do fluxo (admin) Admin
Painel para o admin editar os menus do chatbot (nomes, ordem, adicionar/remover opções) sem precisar de código.
💡 Legenda: Essencial = recomendo incluir sempre   Plus = agrega muito valor   Admin = painel de controle
