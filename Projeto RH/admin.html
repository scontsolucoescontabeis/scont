Regras de Renomeação de Arquivos

1. Padrões de Renomeação

Tags Disponíveis:

  • {CODIGO_EMPRESA} : Extrai o código (DE) ou insere o código (PARA).
  • {NOME_EMPRESA} : Insere o nome da empresa (apenas no PARA).
  • {NOME_ARQUIVO} : Extrai o nome original (DE).
  • {NOME_DOCUMENTO} : Insere o nome mapeado (apenas no PARA). Requer cadastro no passo 2 abaixo.
  • {MM} : Mês da competência (2 dígitos).
  • {AAAA} : Ano da competência (4 dígitos).
  • {IGNORAR} : Ignora uma parte do texto original (apenas no DE).

Exemplo DE: {CODIGO_EMPRESA}_{NOME_ARQUIVO}_{IGNORAR}_{MM}{AAAA}

Exemplo PARA: {NOME_DOCUMENTO}_{NOME_EMPRESA}_{AAAA}_{MM}
